FogBugzLib documentation

Loggin in and out

FogBugz fogbugz = FogBugz.login(server, username, password);
To logout:
fogbugz.logout();

Retrieving cases

Retrieving a particular case:
Case cs = Case.get(fogbugz, 20000);
Listing filters:
List<Filter> filters = Filter.listFilters(fogbugz);
Executing a particular filter:
List<Case> cases = Filter.getByName(fogbugz, "s-Ocasys").execute();

Mailboxes, People, Statuses, Priorities, Projects, Areas and Categories

Listing objects:
List<Mailbox> mailboxes = Mailbox.listMailboxes(fogbugz);
List<Person> people = Person.listPeople(fogbugz);
List<Status> statuses = Status.listStatuses(fogbugz);
...
Note that these objects are cached! If an object is added, deleted or modified, the changes won't be visible in the library until the cache is cleared:
fogbugz.clearCache();
Getting a single object:
Mailbox mailbox = Mailbox.get(fogbugz, ...);
Person person = Person.get(fogbugz, ...);
Status status = Status.get(fogbugz, ...);
...

Some objects have additional getters, like getByName, getByEmail etc.

Changing cases

Edit a case:
cs.edit("Edit message");
When editting a case it is possible to change the category, project, area etc. To do so, first call setCategory, setProject, setArea before calling edit()
When changing the area, don't forget to also update the project

Assigning a case:
cs.setAssignedTo(Person.getByEmail(fogbugz, "user@domain.com"));
cs.assign("blablabla");
